## Regenerar capturas
1. Suba o app local (`npm start` ou `scripts/run-bot.ps1`).
2. Execute:

```powershell
npm run docs:screenshots
```

O script gera automaticamente todos os arquivos `.png` em `docs/screenshots/`.
